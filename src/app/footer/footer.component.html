Sobre

Simulação de um e-commerce de informática, desenvolvido para fins educacionais, utilizando Angular. O projeto tem como objetivo demonstrar a criação de uma aplicação web funcional, com recursos como exibição de produtos, carrinho de compras e integração com redes sociais.
